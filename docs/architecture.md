# .NET + React Developer Portfolio Architecture

## Project Goal

Create a professional portfolio showcasing full-stack developer skills, with a focus on modern frontend (Next.js).

## Workflow at the intersection of mockups

1. **Analysis:** When a user requests a new view, the agent first reads the contents of the `ui_mocks` folder.
2. **Skeleton:** The agent generates the view structure.
3. **Components:** The agent extracts repetitive elements into `/components` using `shadcn/ui`.
